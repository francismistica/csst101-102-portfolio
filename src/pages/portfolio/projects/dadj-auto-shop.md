# Dadj Auto Shop Management System

**Dadj Auto Shop Management System** is a robust web application built to modernize and streamline the daily operations of an auto repair shop. It provides a comprehensive solution for managing service orders, inventory, and customer details, ensuring efficiency and better customer service.

## Key Features

-   **Dashboard Overview**: Get a quick snapshot of daily operations, including active orders and earnings.
-   **Service Order Management**: Create, track, and update service orders with ease.
-   **Inventory System**: Keep track of parts and supplies to prevent stock shortages.
-   **Customer Management**: Maintain a database of customers and their service history.

## Tech Stack

Built with modern technologies for performance and scalability:
-   **Vue.js**: For a reactive and responsive user interface.
-   **Tailwind CSS**: For a clean, modern, and consistent design.
-   **PostgreSQL**: A powerful relational database for secure and reliable data storage.
-   **Vercel**: For fast and reliable deployment.
