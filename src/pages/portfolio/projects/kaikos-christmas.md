# Kaiko's Christmas

**Kaiko's Christmas** is a festive web application designed to spread holiday cheer. It leverages AI to generate unique and personalized Christmas greetings, making it easy to send thoughtful messages to friends and family.

## Key Features

-   **AI-Powered Greetings**: Generates custom messages based on user input.
-   **Festive UI**: A warm and inviting design to capture the holiday spirit.
-   **Social Sharing**: Easily share generated greetings on social media.

![Generated Greeting Preview](/images/projects/xmas-kiko2.png)

## Tech Stack

Built with:
-   **React** for a dynamic user interface.
-   **TailwindCSS** for beautiful, responsive styling.
-   **AI Integration** for content generation.
